
# HMCTS Task Creation App (Starter)

This is a minimal full‑stack starter for the **HMCTS DTS Developer Technical Test – Junior Software Developer**.

It includes:
- **Backend:** Node.js + Express + SQLite, with validation and unit tests
- **Frontend:** React + Vite form to create a task and show confirmation
- **API docs:** Minimal OpenAPI file
